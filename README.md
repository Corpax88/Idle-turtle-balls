# Idle Turtle Balls

Et idle/action-spill laget av Mats + ChatGPT

## Hovedidé
Baller treffer en steampunk-skilpadde for å tjene penger.  
Bossene dropper deler til The Great Steam Engine, som bygges opp i bakgrunnen.

## Nåværende mål
- Gjøre skilpadden mindre
- Flytte Steam Engine bak skilpadden
- Fjerne unødvendig tekst
- Gjøre Hero Ball mer intuitiv

# Idle Turtle Balls v0.30.0 Boss Arena

Et lite idle/clicker-spill bygget som en statisk nettside.

## Filer

- `index.html` - HTML-struktur
- `style.css` - all styling
- `script.js` - all gameplay-logikk

## Publisere med GitHub Pages

1. Last opp `index.html`, `style.css`, `script.js` og `README.md` til et GitHub-repo.
2. Apne repoet pa GitHub.
3. Velg `Settings` -> `Pages`.
4. Under `Build and deployment`, velg:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - mappe: `/root`
5. Trykk `Save`.

Etter litt far du en lenke som ligner:

`https://brukernavn.github.io/repo-navn/`
